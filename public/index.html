About

Hi there! My name is Tiffany Chieu. I'm a student at UCLA majoring in Computer Science and Engineering. My interests include programming languages and compilers, computer vision and image processing, and embedded systems. When I'm not programming, you can find me hiking, reading, or playing the piano.

Currently, I'm interning at JPL on the Small Scale Flight Software group, where I'm developing compiler tools for the FPP modeling language, and integrating it into a suite of automatic code generation tools provided by the F' flight software framework.

Feel free to check out my projects above, or take a look at my github to see more!
